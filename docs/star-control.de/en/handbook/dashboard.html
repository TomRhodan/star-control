Handbook

    Dashboard

    Your command center with live Star Citizen news, server status, and community statistics.

    Status Cards

    The top row displays three status cards that give you a quick overview:

    • Star Citizen Installation - Shows your install path and detected language. A yellow indicator means a translation update is available; green means everything is current. Use the "Open Folder" button to quickly access the installation directory.
    • Wine Runner - Displays the currently selected Wine/Proton runner and its status (ready, not configured, or missing).
    • Launch - A prominent button to start Star Citizen directly from the Dashboard. Disabled when the game is not fully installed.

    RSI News Feed

    The news panel fetches live articles from the Roberts Space Industries website. Each article shows a title, category, timestamp, and brief summary. Click any article to open it in your browser.

    If the news feed fails to load (e.g., no internet connection), a retry button appears to refresh the feed.

    Server Status

    The server status panel monitors the Star Citizen infrastructure and shows the operational status of key components. Status indicators are color-coded:

    • Green - Operational
    • Yellow - Degraded performance
    • Red - Major outage
    • Gray - Unknown

    Community Statistics

    This panel shows real-time community metrics:

    • Total Funding - The total amount pledged by the community
    • Star Citizens - Total number of registered players
    • Vehicles - Number of vehicles in the game

    Below the current numbers, sparkline charts show historical trends over 7-day and 30-day periods. Use the period toggle buttons to switch between views. The delta percentage shows how much each metric has changed.

    All Dashboard data loads in parallel for fast startup. If any panel fails to load, it shows an error state with a retry button - the other panels remain unaffected.
