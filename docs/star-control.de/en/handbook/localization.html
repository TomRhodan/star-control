Handbook

    Localization

    Install community-maintained language packs to play Star Citizen in your preferred language.

    Current Status

    The top section shows your currently installed language pack (if any), including:

    • Language name
    • Commit SHA (version identifier)
    • Installation date
    • Update indicator - shows if a newer version is available

    Available Languages

    Below the status, a list of downloadable community language packs is displayed. Each entry shows:

    • Language name and source repository
    • Install button for languages not yet installed
    • Update button when a newer version is available
    • Remove button for installed languages

    Progress is shown during installation and updates.

    Star Control automatically checks for translation updates when you open the Dashboard. If an update is available, a yellow indicator appears on the installation status card, and you can update with a single click.
    Language packs are community-maintained translations, not official CIG translations. The quality and completeness may vary. Remote language information is cached for up to one hour.
