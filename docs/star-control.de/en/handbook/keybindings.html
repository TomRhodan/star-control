Handbook

    Keybindings

    View and edit controller bindings, manage devices, and fine-tune joystick settings.

    Binding Editor

    The binding editor displays all Star Citizen keybindings organized by device type and action category.

    Device Filters

    Filter bindings by device type using the tabs at the top:

    • Keyboard - Standard keyboard shortcuts
    • Mouse - Mouse button bindings
    • Gamepad - Controller/gamepad bindings
    • Joystick - HOTAS and joystick bindings

    Filtering and Search

    • Action Search - Type to filter bindings by action name in real-time
    • Category Filter - Group bindings by Star Citizen category (Flight, Targeting, Vehicles, etc.)
    • Customized Only - Show only bindings that have been modified from defaults

    Editing a Binding

    Click the edit button on any binding to open the binding editor modal:

    1. The current binding is displayed along with the action name
    2. Select the target device (keyboard, mouse, gamepad, or joystick)
    3. Press the desired key or button to capture the new binding
    4. If the binding conflicts with an existing one, a warning is shown
    5. Click Save to apply or Cancel to discard the change

    Use the Clear button to remove a binding, or Reset to restore the default.

    Joystick Management

    Star Control detects connected joysticks and displays them in a list. You can drag and drop joysticks to reorder their instance numbers, which controls how Star Citizen assigns js1_, js2_, etc. to your devices.

    Joystick instance numbers in Linux can differ from what Star Citizen expects. If your bindings seem to be on the wrong device, try reordering your joysticks here.

    Joystick Tuning

    Fine-tune each joystick's behavior with these settings:

    • Deadzone - Adjust the deadzone slider to ignore small movements around the center position. Useful for older joysticks with drift.
    • Sensitivity Curve - Select a response curve that changes how input translates to in-game movement. Linear gives a 1:1 response; other curves provide more precision near the center.
    • Axis Inversion - Toggle inversion for the X, Y, and Z axes independently.
    • Sensitivity Multiplier - Scale the overall sensitivity up or down.

    Each joystick can be tuned independently, so you can have different settings for your left and right sticks in a dual-stick setup.

    Joystick tuning settings are stored per device. If you swap joystick USB ports, the settings follow the device, not the port.
