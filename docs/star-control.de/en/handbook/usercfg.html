Handbook

    Graphics Settings

    Visual editor for Star Citizen's USER.cfg file with categorized settings and descriptions.

    Overview

    The USER.cfg file controls Star Citizen's graphics, performance, and input settings. Instead of editing this text file manually, Star Control provides a visual editor with categorized settings, value ranges, and descriptions.

    Setting Categories

    Settings are organized in collapsible groups:

    • Essential (always visible) - Resolution, Window Mode, Graphics Renderer (Vulkan/DX11), VSync, Max FPS, TSR, Debug HUD
    • Quality - Overall preset, Object Detail, Particles, Physics, Shading, Shadows, Textures, Water
    • Shaders - FX, General, Post Process, Shadow, Glass, Particle shader quality
    • Textures - Texture streaming and pooling settings
    • Effects - Post-processing, bloom, color correction
    • Clarity - Anti-aliasing and temporal adjustments
    • LOD - Level of detail distances
    • Input - Mouse and controller sensitivity, dead zones
    • Advanced - Memory, debug, and experimental settings

    Control Types

    Each setting uses the appropriate control for its value type:

    • Slider - For numeric ranges (e.g., Max FPS: 0-300 with 5-step increments)
    • Toggle - For on/off settings (e.g., VSync)
    • Dropdown - For preset choices (e.g., Low / Medium / High / Very High)
    • Text Input - For free-form values (e.g., resolution in WIDTHxHEIGHT format)

    Each setting displays its current value, allowed range, and a description explaining what it does.

    Saving Changes

    Changes are tracked against the last saved state. Use the buttons at the top to manage your settings:

    • Save - Writes all changes to the USER.cfg file
    • Revert - Discards unsaved changes and restores the last saved state
    If the USER.cfg file is modified outside of Star Control (e.g., by the game itself), a warning appears. You can choose to reload the external changes or keep your current values.
    Start with the Essential category. Setting the Graphics Renderer to Vulkan and configuring Max FPS can significantly improve your experience. The Quality settings let you fine-tune the balance between visual fidelity and performance.
