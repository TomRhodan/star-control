Handbook

    Getting Started

    Download Star Control, launch it for the first time, and walk through the Setup Wizard.

    Download Options

    Star Control is available in three formats from the GitHub Releases page:

    • DEB Package - For Debian, Ubuntu, Linux Mint, and derivatives. Install with sudo dpkg -i star-control_*.deb.
    • AppImage - Portable, works on any Linux distribution. Make executable with chmod +x Star-Control_*.AppImage and run.
    • Portable ZIP - Extract and run directly. No installation needed.

    First Launch

    When you start Star Control for the first time, the application will:

    1. Create its configuration directory at ~/.config/star-control/
    2. Detect your system language and set the UI accordingly (English or German)
    3. Open the Dashboard as your starting point
    If you have previously used LUG-Helper or installed Star Citizen manually, Star Control can detect existing installations. Check the Settings page to verify your base path is correctly configured.

    The Setup Wizard

    The recommended way to set up Star Citizen with Star Control is through the Installation page, which guides you through the complete process in three steps.

    Step 1: System Compatibility Check

    Star Control verifies your system meets the requirements for Star Citizen on Linux:

    • Memory (RAM) - Checks available system memory
    • AVX Support - Verifies CPU supports AVX instructions
    • Inotify Mapcount - Checks the kernel parameter vm.max_map_count
    • File Descriptor Limit - Verifies the open file limit is sufficient
    • Vulkan Support - Confirms Vulkan graphics API is available
    • Disk Space - Checks available storage

    Click Run Check to execute all tests. Each check shows a pass or fail result with details. You can proceed to the next step once all checks pass.

    If the inotify mapcount check fails, you may need to increase the value. The LUG Wiki has instructions for your distribution. Star Control cannot change kernel parameters for you.

    Step 2: Configuration

    Configure your installation paths and download a Wine runner:

    • Install Directory - Choose where Star Citizen will be installed. The Wine prefix and game files will live here.
    • Wine Runner - Select and download a Wine/Proton runner from community sources (LUG, Kron4ek, RawFox, or Mactan).
    • Performance Options - Optionally configure ESync, FSync, DXVK Async, and other launch options.

    Step 3: Installation

    The automated installation runs through these phases:

    1. Prepare - Creates the directory structure and Wine prefix
    2. Winetricks - Installs required Windows dependencies
    3. DXVK - Sets up GPU driver compatibility layer
    4. Registry - Applies required Windows registry settings
    5. Download - Downloads the RSI Launcher
    6. Install - Installs the RSI Launcher into the Wine prefix
    7. Launch - Performs an initial launch to complete setup

    A real-time log shows the progress of each phase. Once complete, you can navigate to the Launch page to start Star Citizen.

    The full installation process may take several minutes depending on your internet connection and system speed. The RSI Launcher will need to download the game files separately after this setup is complete.

    What's Next?

    After installation, explore the other sections of Star Control:

    • The Dashboard shows live news and server status
    • The Launch page lets you configure performance options and start the game
    • The Keybindings page helps manage your controller setup
