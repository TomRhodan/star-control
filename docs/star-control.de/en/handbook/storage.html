Handbook

    Storage

    Manage Star Citizen versions and optimize disk usage with copy or symlink strategies.

    Version Management

    The Storage tab lists all detected Star Citizen versions (channels) on your system:

    • LIVE - The main release version
    • PTU - Public Test Universe (test builds)
    • EPTU - Evocati PTU (early test builds)

    For each version, the page shows the installation path, Data.p4k file status and size, and available disk space.

    Data.p4k Management

    The Data.p4k file contains all Star Citizen game assets and is typically 100+ GB. When running multiple versions (e.g., LIVE and PTU), you have two strategies:

    Copy

    Create a standalone copy of Data.p4k for a version. This uses more disk space but ensures each version is independent. A progress bar shows the copy status with elapsed time and estimated time remaining. The copy operation can be cancelled.

    Symlink

    Create a symbolic link from one version to another's Data.p4k. This saves significant disk space (100+ GB) since both versions share the same file. The game's patcher will update the file as needed.

    If you play both LIVE and PTU, using a symlink for Data.p4k can save you over 100 GB of disk space. The PTU patcher will update the shared file when needed.

    Delete Version

    Remove a Star Citizen version from your system. A confirmation dialog ensures you don't accidentally delete game files.

    Deleting a version removes all its game files permanently. If you used a symlink, only the link is removed - the original Data.p4k remains untouched.
