Handbuch

    Speicherverwaltung

    Star Citizen Versionen verwalten und Speicherplatz mit Kopier- oder Symlink-Strategien optimieren.

    Versionsverwaltung

    Der Speicher-Tab listet alle erkannten Star Citizen Versionen (Kanaele) auf deinem System auf:

    • LIVE - Die Hauptversion
    • PTU - Public Test Universe (Testversionen)
    • EPTU - Evocati PTU (fruehe Testversionen)

    Fuer jede Version zeigt die Seite den Installationspfad, den Data.p4k-Dateistatus und die Groesse sowie den verfuegbaren Speicherplatz.

    Data.p4k-Verwaltung

    Die Data.p4k-Datei enthaelt alle Star Citizen Spiel-Assets und ist typischerweise 100+ GB gross. Wenn du mehrere Versionen nutzt (z.B. LIVE und PTU), hast du zwei Strategien:

    Kopieren

    Erstelle eine eigenstaendige Kopie der Data.p4k fuer eine Version. Das verbraucht mehr Speicherplatz, stellt aber sicher, dass jede Version unabhaengig ist. Ein Fortschrittsbalken zeigt den Kopierstatus mit verstrichener Zeit und geschaetzter Restzeit. Der Kopiervorgang kann abgebrochen werden.

    Symlink

    Erstelle einen symbolischen Link von einer Version zur Data.p4k einer anderen. Das spart erheblich Speicherplatz (100+ GB), da beide Versionen dieselbe Datei teilen. Der Spiel-Patcher aktualisiert die Datei bei Bedarf.

    Wenn du sowohl LIVE als auch PTU spielst, kann ein Symlink fuer Data.p4k dir ueber 100 GB Speicherplatz sparen. Der PTU-Patcher aktualisiert die gemeinsame Datei bei Bedarf.

    Version loeschen

    Entferne eine Star Citizen Version von deinem System. Ein Bestaetigungsdialog stellt sicher, dass du nicht versehentlich Spieldateien loeschst.

    Das Loeschen einer Version entfernt alle ihre Spieldateien dauerhaft. Wenn du einen Symlink verwendet hast, wird nur der Link entfernt - die originale Data.p4k bleibt unberuehrt.
