    Installation

    Ein dreistufiger Assistent, der Star Citizen unter Linux von Grund auf einrichtet.

    Die Installationsseite ist ein schrittweiser Assistent, der alles erledigt, was noetig ist, um Star Citizen unter Linux zum Laufen zu bringen. Eine Fortschrittsanzeige oben zeigt, bei welchem Schritt du bist, und du kannst jederzeit zu vorherigen Schritten zuruecknavigieren.

    Schritt 1: Systemkompatibilitaetspruefung

    Vor der Installation prueft Star Control, ob dein System die Anforderungen erfuellt:

    • Arbeitsspeicher (RAM) - Star Citizen benoetigt viel Speicher. 16 GB sind das Minimum, 32 GB oder mehr werden empfohlen.
    • AVX-Unterstuetzung - Das Spiel erfordert eine CPU mit AVX-Befehlssatzunterstuetzung.
    • Inotify Mapcount - Der Kernel-Parameter vm.max_map_count muss hoch genug gesetzt sein (typischerweise 16777216).
    • Dateideskriptor-Limit - Das Limit fuer offene Dateideskriptoren (ulimit -n) muss ausreichend sein.
    • Vulkan-Unterstuetzung - Eine Vulkan-faehige GPU mit korrekten Treibern ist erforderlich.
    • Speicherplatz - Ausreichend freier Speicherplatz fuer die Installation (Star Citizen benoetigt 100+ GB).

    Klicke auf Pruefung starten, um alle Tests auszufuehren. Ergebnisse bleiben erhalten, wenn du weg navigierst und zurueckkommst.

    Wenn eine Pruefung fehlschlaegt, behebe das Problem, bevor du fortfaehrst. Der Systemcheck kann keine Kernel-Parameter aendern oder Treiber fuer dich installieren - er meldet nur den aktuellen Zustand.

    Schritt 2: Konfiguration

    Konfiguriere, wo und wie Star Citizen installiert werden soll:

    • Installationsverzeichnis - Verwende den Durchsuchen-Button, um das Basisverzeichnis auszuwaehlen. Das Wine-Prefix und die Spieldateien werden in diesem Verzeichnis erstellt.
    • Wine Runner - Waehle und lade einen Wine/Proton Runner herunter. Mehrere Community-Quellen sind verfuegbar (siehe Wine Runners fuer Details).
    • Performance-Optionen - Konfiguriere Startoptionen wie ESync, FSync und DXVK Async vor (diese koennen spaeter auf der Startseite geaendert werden).

    Schritt 3: Automatisierte Installation

    Die Installation durchlaeuft sieben aufeinanderfolgende Phasen:

    1. Vorbereiten - Erstellt die Verzeichnisstruktur und initialisiert das Wine-Prefix.
    2. Winetricks - Installiert benoetigte Windows-Abhaengigkeiten (Schriftarten, Bibliotheken).
    3. DXVK - Installiert die DirectX-zu-Vulkan-Uebersetzungsschicht fuer GPU-Kompatibilitaet.
    4. Registry - Wendet Windows-Registry-Einstellungen an, die der RSI Launcher benoetigt.
    5. Download - Laedt den RSI Launcher-Installer herunter.
    6. Installieren - Fuehrt den RSI Launcher-Installer im Wine-Prefix aus.
    7. Starten - Fuehrt einen ersten Start durch, um das Setup abzuschliessen und zu verifizieren.

    Ein Echtzeit-Log-Panel zeigt detaillierte Ausgaben jeder Phase. Wenn ein Fehler auftritt, enthaelt das Log Diagnoseinformationen.

    Nachdem der Installationsassistent abgeschlossen ist, musst du dich noch im RSI Launcher anmelden und die eigentlichen Spieldateien herunterladen. Star Control kuemmert sich um das Linux-spezifische Setup; der Spiel-Download erfolgt ueber den offiziellen RSI Launcher.
