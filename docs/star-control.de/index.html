✦ The Origin Story

I'm a huge fan of Star Citizen and truly believe in this ambitious project. I've been playing it on Linux for years, relying on the LUG-Helper since it became available. The LUG Wiki has always been my first stop when something wouldn't run quite right.

Then I discovered the SC Launch Configurator from luftwerft.com running on Windows - and I was jealous. Every attempt to get something comparable working reliably on Linux fell short of expectations.

That's when the idea hit: combine all those insights and ideas into a single Linux application. Star Control was born from that vision.

🔧 LUG-Helper 📚 LUG Wiki ⚙️ luftwerft.com

Thank you to everyone in the community who inspired this project. I hope Star Control helps you spend more time playing and less time configuring.

Powerful Features

Everything you need to run Star Citizen on Linux with ease.

📊

Command Center

Live dashboard with RSI news, server status, and community funding stats at a glance.

🚀

Launch Manager

One-click launch with configurable performance options: ESync, FSync, DXVK Async, Wayland, HDR, FSR, MangoHUD.

🛠️

Wine Runner Management

Download and manage Wine/Proton runners from multiple community sources (LUG, Kron4ek, RawFox, Mactan).

🎮

Controller Configuration

View connected devices, keybindings, reorder joystick instances, and manage binding profiles.

📝

USER.cfg Editor

Visual editor for all Star Citizen graphics, performance, and quality settings.

🌍

Localization

Install community translations with one click, with automatic update detection.

💾

Profile Management

Backup and restore Star Citizen profiles (actionmaps.xml, attributes.xml, USER.cfg).

📦

Multi-version Support

Manage LIVE, PTU, EPTU, and other Star Citizen channels from a single interface.

⚡

DXVK Management

Install and update DXVK versions with automatic DLL deployment.

🔧

Prefix Tools

Winecfg, DPI scaling, PowerShell installation via winetricks - all in one place.
